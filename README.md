# JSL 指文字認識 学習版

あなたの動きから学習して、より正確に指文字を認識するプロトタイプです。

## セットアップ

### 1. ローカルサーバー起動
```bash
cd ~/Desktop/JSL_app
ruby -run -e httpd -- -p 8080 .
```

### 2. ブラウザで開く
```
http://localhost:8080
```

### 3. カメラ許可
ブラウザのアドレスバー横でカメラ許可を与えてください。

## 使い方

### 📝 記録モード

1. **「記録モード」** ボタンをクリック
2. **指文字を行う** → 文字名を入力（例：「あ」）→ **「記録」** ボタン
3. **同じ文字を複数回記録**（5～10回推奨）
4. **「📥 データエクスポート」** をクリック → JSON ファイルダウンロード
5. ダウンロードファイルを以下に移動：
   ```
   data/$(date +%Y-%m-%d)/training_data_YYYY-MM-DD.json
   ```

### 🔍 認識モード

1. **「認識モード」** ボタンをクリック
2. **指文字を行う** → 認識結果が表示される
3. 学習データが多いほど精度UP

## 機能説明

| 機能 | 説明 |
|---|---|
| **認識モード** 🔍 | 学習データから指文字を認識。データ不足時はルール判定で対応 |
| **記録モード** 📝 | カメラで指文字を記録。ランドマーク（21個の関節位置）を保存 |
| **距離ベース分類** | 入力を記録済みサンプルと比較。最も距離が近い文字を選択 |
| **フォールバック** | 学習データが不足している場合、従来のルール判定で認識 |
| **永続保存** 💾 | ブラウザの localStorage に自動保存 |
| **エクスポート** | JSON 形式でダウンロード可能。GitHub に統合管理 |

## データ形式

エクスポートされる JSON：
```json
{
  "timestamp": "2026-05-24T12:34:56.789Z",
  "totalSamples": 45,
  "charCounts": {
    "あ": 5,
    "い": 5,
    "う": 5,
    ...
  },
  "data": {
    "あ": [
      [ { "x": 0.5, "y": 0.5, "z": 0 }, ... ],  // ランドマーク21個
      [ { "x": 0.51, "y": 0.49, "z": 0 }, ... ],
      ...
    ],
    ...
  }
}
```

## GitHub 統合

毎日のデータを `data/YYYY-MM-DD/` に保存し、Git で管理：

```bash
# データを記録フォルダに移動
cp ~/Downloads/training_data_2026-05-24.json data/2026-05-24/

# コミット
git add data/2026-05-24/
git commit -m "Add training data for 2026-05-24 (45 samples)"
git push origin main
```

## 認識ルール（フォールバック）

学習データが不足している場合、以下のルールで認識します：

- **あ** : 親指だけ立てる
- **い** : 小指のみ立てる
- **う** : 人差し指＋中指
- **え** : 5本指を曲げる（グー）
- **お** : 親指×人差し指でO字
- **か** : 親指＋人差し指のL字（縦向き）
- **し** : 親指＋人差し指のL字（横向き）
- **き** : 人差し指＋小指
- **く** : 4指のばし（横向き）
- **け** : 4指のばし（縦向き）
- **そ** : 人差し指のみ（横向き）

詳細は `rules/fallback_rules.json` を参照。

## トラブルシューティング

### カメラが動作しない
- ブラウザの再起動
- カメラ許可を確認（Chrome 設定 → プライバシーとセキュリティ）
- HTTPS または `localhost` で実行（ローカルサーバーで OK）

### データが保存されない
- ブラウザの localStorage 確認
- シークレットモードでは動作しません
- DevTools → Storage → Local Storage を確認

### 認識精度が低い
- 同じ文字を複数回（5～10回）記録してください
- 様々な角度や距離から記録してください
- ライティングを一定に保つ
